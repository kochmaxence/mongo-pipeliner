aggregate

Run aggregate on model

(model, options) => (...stagesFn)

aggregate
Since: 0.0.0
Parameters
model (object) A mongoose model or any object with an aggregate method.
Name Description
model.aggregate function The function receiving the pipeline.
options (object)
stagesFn (...function) Stages functions
Returns
object: pipeline result
Example
const pipeline = aggregate(match({ name: 'username' }));
console.log(pipeline);

aggregatePaginate

Run aggregate on model, with an extra step at the end to produce pagination.

(model, options) => (...stagesFn) => (page, limit)

aggregatePaginate
Since: 0.0.0
Parameters
model (object) A mongoose model or any object with an aggregate method.
Name Description
model.aggregate function The function receiving the pipeline.
options (object)
Name Description
options.countField string String used as a key for the count field.
options.resultField string String used as a key for the result field.
options.hasNextField string String used as a key for the hasNext field.
options.hasPrevField string String used as a key for the hasPrev field.
options.nextPageField string String used as a key for the nextPage field.
options.prevPageField string String used as a key for the prevPage field.
options.pagesField string String used as a key for the pages field.
options.limitField string String used as a key for the limit field.
options.pageField string String used as a key for the pages field.
stagesFn (...function) Stages functions
page (number) Current page
limit (number) Limit of documents to return
Returns
object: pipeline result
Example
const getUsers = aggregatePaginate(User)
getUsers([ match({ name: 'username' }) ])(0, 10)
  .then(console.log);
