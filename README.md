# mongo-pipeliner

Yet Another Pipeline Generator... With bonuses.

## Getting started

### Prerequisites

The package `mongodb` is used to cast ObjectId. If you already use a library like `mongoose`, you should already have it.

### Installing

`npm install --save mongo-pipeliner`
